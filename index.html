About This Project

This interactive website was created to make learning about animal cell biology engaging and accessible. The 3D design and glassmorphism effects help visualize complex cellular structures in a modern, intuitive way.

Our goal is to provide an educational resource that can be used by students, teachers, and anyone curious about the microscopic world inside living organisms.
